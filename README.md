# Budget Ripper — Telegram Mini App

Персональний трекер витрат прямо в Telegram. Додавай витрати, встановлюй ліміти за категоріями та стеж за своїми фінансами — все в одному зручному міні-додатку.

---

## Про проект

**Budget Ripper** — це Telegram Mini App для відстеження особистих витрат. Користувач відкриває додаток прямо в месенджері, додає витрати за категоріями, бачить аналітику у вигляді кругової діаграми та контролює ліміти по кожній категорії.

### Що вміє додаток

- Додавати витрати з вибором категорії та описом
- Переглядати загальну суму витрат за поточний місяць
- Бачити розбивку витрат по категоріях у вигляді донат-діаграми
- Встановлювати місячні ліміти на категорії
- Отримувати сповіщення від бота коли ліміт досягає 80% або 100%
- Переглядати детальну аналітику з прогрес-барами

## Стек технологій

### Backend — `backend/`

| Файл | Що робить |
|------|-----------|
| `main.py` | Головний модуль: FastAPI додаток + aiogram бот. Містить всі API ендпоінти, вебхук для Telegram, обробники команд бота та валідацію initData |
| `models.py` | SQLAlchemy моделі: `User`, `Transaction`, `Limit` — структура таблиць бази даних |
| `database.py` | Підключення до PostgreSQL через asyncpg, ініціалізація бази |
| `schemas.py` | Pydantic схеми для валідації запитів та відповідей API |
| `requirements.txt` | Список Python залежностей |

**Технології:** Python 3.12, FastAPI, aiogram 3, SQLAlchemy 2.0 (async), asyncpg, PostgreSQL, Pydantic v2

### Frontend — `frontend/`

| Файл | Що робить |
|------|-----------|
| `index.html` | Розмітка додатку: 4 екрани (Головна, Додати витрату, Аналітика, Ліміти), модальне вікно, навігація |
| `styles.css` | Весь дизайн: темна тема, glassmorphism, aurora-градієнти, SVG навігація, анімації |
| `app.js` | Вся логіка: запити до API, рендер транзакцій, донат-діаграма на Canvas, навігація між екранами |

**Технології:** Vanilla HTML/CSS/JS, Telegram WebApp SDK, Canvas API
